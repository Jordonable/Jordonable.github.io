[105] 从前序与中序遍历序列构造二叉树

Created2026-03-30|Updated2026-03-30
|Post Views:

核心思路

利用前序和中序遍历的特性还原二叉树：

  1. 前序遍历第一个元素是根节点
  2. 在中序遍历中找到根节点位置，可以分割左右子树
  3. 递归构建左右子树

关键洞察：前序确定根，中序确定边界。

代码实现

关键点

1. 区间分割逻辑

在 [preStart, preEnd] 和 [inStart, inEnd] 中：

  • preorder[preStart] 是当前子树的根
  • n = index.get(preorder[preStart]) 是根在中序中的位置
  • 中序左子树：[inStart, n-1]，右子树：[n+1, inEnd]

2. 前序区间分割（重点）

  • leftSize = n - inStart 计算左子树节点数
  • 前序左子树：[preStart+1, preStart+leftSize]
  • 前序右子树：[preStart+leftSize+1, preEnd]

3. HashMap 优化

提前建立中序值到索引的映射，避免每次递归都 O(n) 查找根节点位置。

易错点

  1. 索引计算错误：leftSize = n - inStart 而不是 n
  2. 边界条件：递归终止条件是 preStart > preEnd 或 inStart > inEnd
  3. 区间偏移：前序右子树起点是 preStart + leftSize + 1，容易漏掉 +1

复杂度分析

  • 时间复杂度：O(n)，每个节点访问一次，HashMap 查找 O(1)
  • 空间复杂度：O(n)，HashMap 存储 + 递归栈深度 O(h)，最坏 O(n)

原理总结

三种遍历的任意一种 + 中序遍历都可以唯一还原二叉树：

  • 前序 + 中序 ✓（本题）
  • 后序 + 中序 ✓
  • 层序 + 中序 ✓
  • 前序 + 后序 ✗（无法唯一确定，除非是满二叉树）
